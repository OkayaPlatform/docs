# Integrate Okaya with OneLine

<CodeGroup>


```python Python
    from okaya import Okaya, OkayaConfiguration

    # Usage:
    config = OkayaConfiguration()
    okaya_handler = Okaya(config)
    # okaya_handler.validate_configuration()


    text_to_sentiment = "I love ice cream ! "
    sa_response = okaya_handler.run(prompt_name='santiment-analysis',
                                  variables={'TEXT': text_to_sentiment,})
```

```javascript JS
    import { Okaya, OkayaConfiguration } from 'okaya';

    const config = new OkayaConfiguration();
    const okayaHandler = new OneLine.Okaya(config);
    // okayaHandler.validateConfiguration(); // Uncomment if needed

    const textToSentiment = "I love ice cream!";
    const sentimentAnalysisResponse = okayaHandler.run('santiment-analysis', {
      TEXT: textToSentiment,
    });
```
